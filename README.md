## 细颗粒物 检测
细颗粒物，尤其是PM2.5 是指环境空气中直径小于2.5微米的颗粒物，它能够较长时间的悬浮于空气中，尤其是在空气中含量浓度越高，就代表着空气污染越来越严重。
这些颗粒物达到足够的浓度和时间可能会因此危害人类的健康和生活环境。
